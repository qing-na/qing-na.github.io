阿里云 OSS

对象存储（英语：Object storage）是一种计算机数据存储架构，它将数据作为对象进行管理，与其他存储架构不同（如文件系统将数据作为文件层次结构进行管理，而块存储则将数据作为扇区和轨道内的块进行管理）。每个对象通常包括数据本身、数量不等的元数据和一个全局唯一的标识符。

购买阿里云 OSS 服务

  1. 打开网站：阿里云对象存储 OSS
  2. 选择套餐：
    • 商品类型：OSS 资源包
    • 资源包类型：标准 - 本地冗余存储
    • 地域：中国内地通用
    • 购买时长：6 个月

创建 Bucket

在对象存储 OSS 控制台创建一个 Bucket

填写必要的信息，注意要选择“公共读”
其中 Buket 名称必须全局唯一

获取 AccessKey

  1. 点击头像，选择 AccessKey 管理
  2. 保存 AccessKey ID 和 Accesskey Secret

    务必保存好，AccessKey ID 和 Accesskey Secret 只能在创建时查看

  3. 再次进入 AccessKey 管理页面，选择使用子用户 AccessKey，避免潜在安全风险
  4. 授予子用户权限
  • AliyunOSSFullAccess——管理对象存储服务（OSS）权限
  • AliyunOSSReadOnlyAccess——只读访问对象存储服务（OSS）的权限

PicGo 配置

  1. 下载安装 PicGo：PicGo
  2. 先进入对应阿里云 OSS 的配置界面，填入必要的信息，信息在之前阿里云 OSS 配置过程中都有
  3. 按照使用习惯设置上传细节

Obsidian 图床设置

下载插件即可使用：image-auto-upload-plugin
