Butterfly 主题

安装

进入 Hexo 根目录，使用 git 下载主题

  • 稳定版
  • 测试版

升级：在主题目录下，运行 git pull

插件安装

至少需要 pug 以及 stylus 的渲染器

配置文件

Hexo 配置文件：

Hexo 文件夹根目录下 _config.yml

Butterfly 主题配置文件

防止升级主题后出现问题，建议在 hexo 的根目录创建一个文件 _config.butterfly.yml，并把主题目录的 _config.yml 内容复制到 _config.butterfly.yml

不是 hexo 的 _config.yml

以后只需在 _config.butterfly.yml 进行配置，Hexo 会自动合并主题 _config.yml 和 _config.butterfly.yml 里的配置，若存在同名，会优先使用 _config.butterfly.yml

系列文章

在文章的 front-matter 上添加参数 series，并给予一个标识
使用此标签外挂，会把相同标识的文章以列表的形式展示
如果不写 series 标识，则默认为你使用此标签外挂所在的文章的 series 标识

Note（Bootstrap Callout）

本主题不建议使用引用块 >，显示效果不佳
推荐使用 Bootstrap 提示框样式

名称 用法
class 【可选】标识，不同的标识有不同的配色
（ default / primary / success / info / warning / danger ）
no-icon 【可选】不显示 icon
style 【可选】可以覆盖配置中的 style
（simple/modern/flat/disabled）

modern
